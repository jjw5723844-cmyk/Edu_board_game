# 🎮 유아~초등학생을 위한 보드게임 천국 (Board Game Heaven)

C# 기반으로 개발된 유아 및 초등학생 대상의 문제 해결력 증진 교육용 보드게임 패키지 프로그램입니다. 현대 교육용 보드게임들을 참고하여 학습 효과를 극대화할 수 있도록 구현되었으며 , 학습자뿐만 아니라 성인 교육자도 함께 참여하여 교감할 수 있도록 설계되었습니다.

## 1. 프로젝트 개요

### 💡 추진 배경

* 
**학습 격차 해소**: 다양한 이러닝 학습 프로그램의 보급에도 불구하고 학습자의 경험과 숙련도에 따른 차이가 여전히 존재합니다.


* 
**기초 프로그램 수요**: 간단한 동작을 통해 학습자가 스스로 과업을 성취할 수 있는 직관적인 학습게임이 지속적으로 필요한 상황입니다.


* 
**교육자 숙련도 향상**: 안내자 역할을 하는 교육자들이 프로그램의 기초 작동 방식을 숙지하고, 학습자의 눈높이에 맞추어 학습을 수행할 수 있도록 돕습니다.



### 🎯 개발 목표

* 
**세대 간 공동 참여**: 유아 및 초등학생을 주 대상으로 하되, 성인 교육자나 관리자도 함께 참여할 수 있는 프로그램을 지향합니다.


* 
**자기주도적 규칙 이해**: C# 기반의 쉬운 동작 구현을 통해 학습자가 스스로 게임의 원리를 터득하도록 유도합니다.


* 
**이론적 기반 설계**: 교육 및 학습 이론의 검증을 거친 현대 보드게임들의 메커니즘을 프로그램 내에 체계적으로 반영합니다.

## 2. 주요 기능 및 게임 소개

본 프로그램은 학습 단계별로 논리, 기억, 공간 능력을 자극하는 3가지 보드게임으로 구성되어 있습니다. 메인 화면(`MainForm`)은 상단 스코어바와 하단 알림 안내 레이아웃이 공통화된 구조를 가집니다.

❶ 하노이의 탑 (Hanoi Tower) — [논리 단계] 

* 
**개요**: 수학적·공학적 사고력과 규칙성 인지 능력을 증진시키는 퍼즐 게임입니다.


* 
**구성**: 3개의 원통형 봉, 서로 다른 넓이를 가진 8개의 원반.


* **게임 규칙**:
1. 원반은 항상 기둥 위에만 놓을 수 있습니다.


2. 한 번에 한 개의 원반만 옮길 수 있습니다.


3. 큰 원반을 작은 원반 위에 올릴 수 없습니다.




* 
**구현 특징**: Stack 구조를 통한 원반 제어 알고리즘 및 이동 횟수 데이터 연동.



❷ 치킨 차차 (Chicken ChaCha) — [기억 단계] 

* 
**개요**: 규칙을 익히고 기억력을 자극하며, 교육자와 학습자 간의 정서적 교감을 유도하는 게임입니다.


* 
**구성**: 닭 말 4마리(빨강, 노랑, 파랑, 초록), 꼬리 깃털 4색, 달걀 모양 트랙 타일 24장, 팔각형 마당 타일(메모리 카드) 12장.


* **게임 규칙**:
1. 
**그림 맞추기**: 내 닭의 바로 앞 칸에 그려진 그림을 마당 타일 중에서 찾아 뒤집습니다.


2. 
**전진과 보너스**: 그림을 맞추면 한 칸 전진 후 한 번 더 기회를 얻고, 틀리면 차례가 넘어갑니다.


3. 
**깃털 빼앗기**: 바로 앞의 닭을 추월하면 그 닭의 꼬리 깃털을 모두 빼앗아 오며, 모든 깃털을 모으면 승리합니다.




* 
**구현 특징**: 동적 함수를 활용한 게임판 생성 및 정·오답 데이터의 델리게이트 콜백 처리.



❸ 우봉고 (Ubongo) — [공간 단계] 

* 
**개요**: 제한 시간 내에 퍼즐 조각을 맞추는 경쟁 게임으로, 판단력과 순발력을 기를 수 있습니다.


* 
**구성**: 양면 퍼즐판, 12개의 퍼즐 조각, 보석 총 18개(사파이어 9개, 호박 9개).


* **게임 규칙**:
1. 
**퍼즐 맞추기**: 주사위를 굴려 지정된 3~4개의 타일을 선택한 후, 제한 시간 내에 빈칸을 남김없이 채웁니다.


2. 
**우봉고 외치기**: 제한 시간(모래시계) 안에 가장 먼저 퍼즐을 완성한 사람이 "우봉고!"를 외치고 보석을 획득합니다.


3. 
**보석 점수 계산**: 총 9라운드 진행 후, 수집한 보석의 색상별 가치 점수를 합산하여 최고점자가 승리합니다.




* 
**구현 특징**: 2D 격자 배열 제어 및 퍼즐 조각의 90도 회전 매트릭스 알고리즘 구현.

## 3. 기술적 특징 (Architecture)

모든 게임의 상태 변화(알림, 점수 갱신, 게임 종료)를 델리게이트 체인(Delegate Chain)으로 일관되게 제어하여 객체지향적 설계 구조를 완성했습니다.

* 
**UI와 로직의 철저한 분리**: 모든 게임 엔진은 직접 화면 컴포넌트를 수정하지 않고 델리게이트 신호만 발동시키며, 부모 폼이 신호를 받아 화면과 소리를 제어하도록 독립성을 유지합니다.


* 
**규격 표준화 및 멀티캐스트**: 알림 및 데이터 갱신용 델리게이트 매개변수를 하나로 통일하여, 단 한 번의 호출로 화면 표시(라벨 변경 등)와 콘솔 로그 출력이 동시에 실행되는 확장성을 확보했습니다.


* 
**안전한 프로그래밍 (Crash 방지)**: 델리게이트를 호출할 때는 반드시 Null 조건부 연산자(`?.`)를 사용하여 연결 오류로 인한 프로그램 불시 종료를 원천 차단합니다.


* 
**유아 중심 폴리싱 및 디버깅 연동**: 오류 메시지 발생 시 유아의 눈높이에 맞춘 친절한 텍스트로 가공하여 전송하며, 교수님 확인용 배포 빌드를 위해 콘솔 창 디버그 로그 출력을 완벽히 연동했습니다.

## 4. 기대 효과

* 
**낮은 진입 장벽**: 교육 현장에서 검증되고 친숙한 보드게임들로 구성되어 학습자와 교육자 모두 빠르게 적응할 수 있습니다.


* 
**차별화된 이러닝 경험**: 실물 보드게임과는 다른 디지털 이러닝 인터페이스만의 고유하고 색다른 몰입감을 제공합니다.


* 
**자기주도적 성취감**: 복잡한 안내 없이도 학습자 스스로 조작하여 보드게임의 과업을 달성하도록 유도합니다.


* 
**유대감 증대 효과**: 성인 교육자와 아이가 함께 게임에 참여해 문답을 주고받음으로써, 일상적인 교감 속에서 유대감을 자연스럽게 높일 수 있습니다.
